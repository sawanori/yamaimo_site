# 山芋の多い料理店 - ウェブサイト構築スキル

## 概要

「山芋の多い料理店」（川崎店・西葛西店・3号店）のコーポレート＋ブランドサイトを、HTML/CSS/JSでゼロから構築する。デザインリファレンスは https://www.marco.tokyo/ だが、テイストは異なる（白ベース・カジュアル・モダン）。

## 実装前に必ず読むファイル

1. **references/design-system.md** — カラー、タイポグラフィ、スペーシング、コンポーネント仕様
2. **references/site-structure.md** — 全ページ構成、セクション順序、ナビゲーション設計
3. **references/content-data.md** — 全店舗のコンテンツデータ（店舗情報、メニュー、コピー）
4. **references/interaction-spec.md** — アニメーション、スライダー、レスポンシブ仕様

## 技術要件

- **HTML5 + CSS3 + Vanilla JS**（フレームワーク不使用、どのプラットフォームにも移植可能）
- **レスポンシブ**: モバイルファースト、ブレークポイント 375px / 768px / 1024px / 1440px
- **画像**: ダミー画像はプレースホルダーdiv（背景色＋テキスト）で表現。実写真は後差し替え前提
- **フォント**: Google Fonts CDN読み込み
- **アニメーション**: CSS transitionベース、IntersectionObserver でスクロールトリガー
- **外部ライブラリ**: Swiper.js（スライダー用）のみCDN許可。他は原則不使用
- **SEO**: セマンティックHTML、構造化データ（LocalBusiness）、meta description
- **パフォーマンス**: lazy loading、CSS/JS分割なし（単一ファイル構成でOK）

## デザインの鉄則

このサイトは「人間が手間暇かけて作ったデザイン」であることが最優先。以下を厳守すること。

### AIっぽさを排除する具体的なルール

1. **均等配置の禁止** — カードやセクションを均等グリッドに並べない。非対称レイアウト、オフセット、意図的な余白の不均一を使う
2. **テンプレ感の排除** — 全セクションが同じ構造の繰り返しにならないこと。各店舗紹介は同じ情報でもレイアウトを微妙に変える
3. **タイポグラフィの緊張感** — 見出しサイズにコントラストをつける。70px超の大見出しと14pxの本文が同居してよい
4. **写真の扱い** — 写真は「枠の中に収まる」だけでなく、セクションをはみ出す、重なる、マスクされる、など動きのある配置を含める
5. **余白の呼吸** — セクション間のpaddingは一律ではなく、コンテンツの重みに応じて80px〜200pxの間で変動させる
6. **色のアクセント** — 白ベースだからこそ、ワンポイントの色が効く。和のアクセント色（山芋の土色、自然薯の肌色、緑）を計算して配置
7. **テクスチャと質感** — べた塗りの白ではなく、微細なノイズテクスチャや紙のような質感をCSS背景で実現する
8. **スクロール体験** — ページ全体を通して「読み進める楽しさ」があること。視差効果やフェードインのタイミングで緩急をつける

### マルコとの差別化ポイント

- マルコ: ダーク背景、酒場の温もり、手書きロゴPNG → **山芋**: 白ベース、素材感・土の温もり、明るく開放的
- マルコ: 縦積みカード一律構成 → **山芋**: セクションごとにレイアウトが変わる、予測不能な構成
- マルコ: 予約導線がフッター固定 → **山芋**: スティッキーCTAが文脈に応じて表示切替
